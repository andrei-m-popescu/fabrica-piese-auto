# Fabrica de Piese Auto

Windows Forms desktop application for managing an auto parts factory inventory,
built with C# .NET Framework and SQL Server.

## Features
- Full CRUD for Products, Orders and Production Lots
- OOP design: ICloneable, IComparable, custom interfaces, operator overloading
- SQL Server LocalDB with parameterized queries
- Custom bar chart drawn with GDI+ (no external libraries)
- Export data to TXT files and Print support

## Requirements
- Visual Studio 2022
- .NET Framework 4.7.2
- SQL Server LocalDB

## How to Run
1. Clone the repository
2. Open `FabricaPiese_PAW_Proiect.sln` in Visual Studio
3. Create a SQL Server LocalDB database named `FabricaPieseAutoDB`
4. Build and run

## Tech Stack
C# · .NET Framework 4.7.2 · Windows Forms · SQL Server LocalDB · GDI+
